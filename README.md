# Projet fin d'année: Agora

## Technos utilisées

- React (webpack inclut)
- Client side rendering
- Mobile first
- PWA
- librairie: ChartJS

## Comment installer le projet ?

installer le projet:  
`npm install`

démarrer le projet:  
`npm run start`
